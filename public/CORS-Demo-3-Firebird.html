CORS Demo 3

Fetching Oxford Dictionary Data via a NodeJS Relay on Firebird

This file uses JavaScript fetch() to acquire and display JSON data from Oxford Dicitonary. Since the Oxford API does not include CORS headers, we have to add them ourselves. This script connects to a NodeJS relay app running on Firebird. The app makes the request on our behalf and forwards it with the appropriate CORS headers included. Working in this way has the added benefit of keeping our API key and password safely hidden from the front end.
